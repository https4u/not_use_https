# 发现一个钓鱼网站

吃瓜群众要擦亮眼睛啊！英文的钓鱼邮件多。

（软广告）有HTTPS的绿标也一定是正牌网站，没事了到我们 [https://myssl.com](https://myssl.com) 来检测一下！

*我们不做网站认证，我们只认CA证书*
